# PULSE SYNTH: 4D Dimension Engine

Pulse Synth is an immersive, high-octane 4D audio-visual synthesizer that blends cyberpunk aesthetics with meme culture. It provides a reactive environment where sound and visuals are strictly linked to user input and dimension scales.

---

## Features

- **10 Dimensions of Sound:** Navigate through 10 distinct audio-visual levels, each with its own frequency range, pulse behavior, and visual distortion.
- **Mood-Reactive Processing:** Type keywords into the mood terminal to instantly recalibrate the synth engine's tempo, filters, and waveforms.
- **Beat-Synced Visuals:** Features a curated collection of unique, rhythm-reactive GIFs that dance and glow in synchronization with the audio pulses.
- **Cyberpunk UI:** Built with a 4D perspective container, CRT scanlines, neon glitch effects, and a glassmorphism control panel.
- **Professional Synth Engine:** Powered by the Web Audio API with support for Square, Sawtooth, Sine, and Triangle waveforms with dynamic low-pass filtering.

---

## Tech Stack

- **Audio:** Web Audio API (Oscillators, BiquadFilters, Gain Nodes)
- **Styling:** Tailwind CSS and Custom CSS Keyframe Animations
- **Visuals:** SVG Integration and Giphy API
- **Deployment:** Vercel or GitHub Pages

---

## How to Play

1. **Enter the Grid:** Click the ENTER THE GRID button to initialize the Audio Context.
2. **Choose your Dimension:** Use the slider or the Up/Down arrows on your keyboard to switch between Dimensions 1–10.
3. **Set the Mood:** Type into the Set Mood box. Try keywords like Spooky, Chill, Intense, or Fun.
4. **Sync/Stop:** Hit the Spacebar to toggle the party on and off.
